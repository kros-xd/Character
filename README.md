# WEB114 Week 2 Assignment: Interactive Character Creator

## Description

This project is a simple interactive character creator built with JavaScript. The user creates a hero for the floating sky city of Aerova by answering a series of prompts.

## Concepts Used

- `alert()` to display messages
- `prompt()` to collect user input
- `confirm()` for yes/no choices
- Template literals (`` `${}` ``) to insert variables into strings
- `parseInt()` to convert user input from a string to a number

## Features

- Create a character name
- Choose a companion
- Select a superpower
- Enter years of training
- Calculate a power level based on training years
- Display a final character summary
